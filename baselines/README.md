# Benchmarks

Generated by `npm run bench:baseline`. Contains absolute file paths so comparison is local-machine only.

## Commands

| Command               | Description                                          |
| --------------------- | ---------------------------------------------------- |
| `npm run bench`       | Run benchmarks (stdout only)                         |
| `npm run bench:baseline` | Run and save results to `baselines/benchmark.json`   |

**Compare against baseline:**
```sh
npm run bench -- --compare baselines/benchmark.json
```

**Update baseline after performance work:**
```sh
npm run bench:baseline
```
